    Network

    Firewall

    UFW is the default firewall for Debian distributions

    • Turn it on with the defaults:
      • sudo ufw enable
    • Check the status
      • sudo ufw status verbose
    • Allow web server
      • sudo ufw allow 80/tcp
      • sudo ufw allow 443/tcp

    iptables is the old standard for firewalls

    nftables is the default for the future
