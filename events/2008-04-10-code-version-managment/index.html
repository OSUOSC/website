Code Version Managment

    We are meeting Thursday, April 10th, at 7:00 in Dreese Labs room 266. The topic for our meeting is source code version control, and what works best for, and there has already been much traffic over this on our mailing list. Subtopics so far include bitkeeper, cvs, git, Google Code, Mercurial, svn, trac, and open discussion of any other source code management programs not yet mentioned that deserve recognition.
